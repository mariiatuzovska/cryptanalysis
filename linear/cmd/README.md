# linear cryptanalysis of Heys cipher command line client

```
NAME:
   linear - linear cryptanalysis of Heys cipher command line client

USAGE:
   cmd [global options] command [command options] [arguments...]

VERSION:
   0.0.1

DESCRIPTION:
   linear cryptanalysis of Heys cipher

AUTHOR:
   Tuzovska Mariia

COMMANDS:
   e        encrypt
   d        decrypt
   search   search for linear approximations
   show     shows approximations that has been found
   attack   finds keys for all approximation alpha and beta in community/approximations.json
   keys     shows keys that has been found for some aplpha and beta
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
