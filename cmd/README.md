# command-line client for differential & linear cryptanalysis of Heys cipher

```
NAME:
   cryptanalysis - cryptanalysis pkg for Heys cipher command line client

USAGE:
   main [global options] command [command options] [arguments...]

VERSION:
   0.0.1

DESCRIPTION:
   linear&differential cryptanalysis for Heys cipher

AUTHOR:
   Tuzovska Mariia

COMMANDS:
   keygen, k, key, kgen     Generate key file for Heys cipher
   encrypt, e, enc          Encrypt file using Heys cipher
   decrypt, d, dec          Decrypt file using Heys cipher
   differential, diff, dif  Differential search
   help, h                  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version

COPYRIGHT:
   2020, mariiatuzovska
```
